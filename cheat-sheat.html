Basic HTML tags

HTML tags name description
<h1> Heading This is used for text heading, h1 is the bigest and h6 is the smallest
<p> Paragraph This is used for creating paragraph on the page
An example of paragraph is <p>This is a paragraph</p>
<b> Bold Make text between tags bold
<i> Italics Make text between tags italics
<div> Division Divide up page content into sections, and applying styles
<table> Table In some points, you need to display some data in a tabular form in your website, the solution is to use <table> tag. Example of table is
<table> opening and closing tag </table>
<tr> Table Row The table row container
<th> Table Header The set of rows defining the column headers in a table
<td> </td> Table Data Table Cell within table row
<html>
<body>
<!--content--> </body>
</html> 		HTML structure All the basic HTML tags must be within the body tags


Table Attributes

Attributes Name Description
colspan=" " Column Span Defines how many columns a table data (td) should span.
rowspan=" " Row Span Defines how many rows a td element should span.


CSS Selector and its properties and values

selector property value description
body text-align: center; To centralize all the text in the document
div background-color: blue; To set the background color of the <div> container to blue.
p color: green; To set the text color of the paragraph to green
